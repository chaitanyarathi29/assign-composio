The Patterns — Read This First

🟢 OAuth2 dominates — 65%

OAuth2 supported by 65+ apps. Build your Composio connector framework here first and you cover the majority of the landscape. API Key is the clear second (40+ apps).

🔵 Developer/Infra is easiest — 80% coverage

GitHub, Vercel, Cloudflare, Supabase, Neo4j, Snowflake, Datadog, Sentry all buildable today. Developer-first docs, free tiers, highest MCP adoption.

🟡 Self-serve is the norm — 76%

76 of 100 apps have free tier or trial with immediate API access. The main blocker for most "not in Composio" apps is that the toolkit hasn't been built yet.

🔴 E-commerce & Finance hardest

E-commerce has platform diversity (WooCommerce self-hosted, no central auth). Finance has the most gated access — Plaid, PitchBook, Paygent require sales or compliance review.

🟣 MCP is emerging — 28 apps

28 apps have an MCP server (official or community). Heaviest adopters: GitHub, Cloudflare, Sentry, Datadog, Supabase, Snowflake, Salesforce, HubSpot, Shopify, Jira.

⚡ Top blocker: Not in registry

42 apps absent from Composio. ~30 have full public REST APIs + free dev access — they're gap opportunities. Real blockers: 10 enterprise-gated + 2 CLI-only tools.
